# Design System — Raising Star Africa

## Brand
- **Name:** Raising Star Africa Foundation
- **Tagline:** Promoting Inclusive Education
- **Description:** Non-governmental organization founded to support, empower and inspire quality education and life skills among differently-abled and less privileged children across Africa.

## Components
### Navigation
- Sticky header with logo on left
- Horizontal menu with dropdowns
- Mobile hamburger menu
- Social media icons in header
- Donate button (prominent CTA)

### Hero Section
- Full-width slider with images
- Text overlay with heading and subheading
- Call-to-action buttons

### Cards
- Project cards with image, title, and description
- Blog post cards with featured image
- Partner/sponsor logo grid

### Buttons
- Primary: Filled with primary color
- Secondary: Outlined or secondary color
- Size variants: Small, Medium, Large

### Footer
- Multi-column layout
- Contact information
- Social media links
- Newsletter signup
